Customizing.

Some template settings that will easily help you customize it.

You can find more detailed and complete documentation of the Figma at this link.

Replacing Images

Select the image you want to replace and drag and drop the image file over the image preview in Fill mode on the interface panel on the right:

How to quickly change a component

If you want to quickly replace a component, then you can do this by going to the Instance section

How to Export SVG

To export vector graphics to SVG you need to click on the "+" symbol in the Export section and select the svg format from the list

How to Export CSS

To get the CSS code of the required element / layer: select the layer and go to the Code tab on the right side of the interface.

How to change color

To quickly change any color in all template files, go to the panel on the right and in the Color Styles list select the color you want to replace and click on the icon to the right of the color to proceed to color replacement.

Edit text layers

To make changes to the content of a Text box, you will need to enter Text Edit mode. You can enter text edit mode in a couple of ways:

  • Double-click on the text layer.
  • Select the text layer and press the Enter or Return key.
